# AI 문서 검증 시스템 v5.1 사용 가이드

## 🎯 v5.1 새 기능 사용법

### 1. 34개 검증 규칙 상세 확인하기

#### 결과 화면에서 확인
분석이 완료되면 결과 화면에 다음과 같이 표시됩니다:

```
[ 34개 검증 규칙 상세 결과 ]
----------------------------------------------------------------------
  1. ✅ 주택매도신청서 존재 여부
  2. ✅ 주택 소재지 기재 확인
  3. ✅ 공고일 이후 작성일
  ...
 30. ❌ 시험성적서_납품확인서
      → 필요 항목 미비: 가스유해성 시험 (열전도율 시험 제외)
  ...
----------------------------------------------------------------------
통과: 33개 | 보완 필요: 1개
```

#### 해석 방법
- ✅ = 검증 통과 (문제 없음)
- ❌ = 검증 실패 (보완 필요)
- 실패한 규칙 아래에 구체적인 사유가 표시됩니다
- 하단에 통과/보완 필요 개수가 집계됩니다

### 2. 시험성적서 정밀 검증 활용하기

#### 시험성적서 검증 항목

분석 결과에서 다음 섹션을 확인하세요:

```
[시험성적서 검증 (규칙 30)]
  ✅ 열방출시험: 포함됨
  ❌ 가스유해성 시험: 미포함 (보완 필요)
  ✅ 납품확인서: 제출됨
  📄 시험성적서 확인된 자재: 폴리우레탄폼, 압출법보온판
```

#### 각 항목 의미

1. **열방출시험**
   - 총열방출량 (THR, Total Heat Release) 시험
   - ✅ = 시험 결과 포함됨
   - ❌ = 시험 결과 없음 → 열방출시험 자료 보완 필요

2. **가스유해성 시험**
   - Gas Toxicity, 가스독성 시험
   - ✅ = 시험 결과 포함됨
   - ❌ = 시험 결과 없음 → 가스유해성 시험 자료 보완 필요

3. **납품확인서**
   - 자재 납품 확인 서류
   - ✅ = 제출됨
   - ❌ = 미제출 → 납품확인서 보완 필요

4. **확인된 자재**
   - 시험성적서에서 인식된 자재명 목록
   - 이 자재들에 대한 시험 결과가 있다는 의미

#### 특수 케이스: 석재 예외

```
[시험성적서 검증 (규칙 30)]
  ℹ️  외벽 마감재가 석재로 확인됨 (시험성적서 생략 가능)
  ✅ 납품확인서: 제출됨
```

- 외벽 마감재가 석재인 경우 시험성적서 없이 납품확인서만 있으면 됩니다
- 이 경우 열방출시험, 가스유해성 시험 항목이 표시되지 않습니다

## 🔍 검증 결과 해석 팁

### 1. 규칙 번호 의미

규칙은 1번부터 34번까지 순차적으로 검증됩니다:

- **1-10번**: 주택매도신청서, 임대현황 관련
- **11-20번**: 위임장, 개인정보동의서 등 양식 서류
- **21-28번**: 등기부등본, 대장 등 정부 발급 서류
- **29-30번**: 준공도면, 시험성적서 (기술 서류) ⭐
- **31-34번**: 세부 검증 항목

### 2. 보완 우선순위

**높음 (즉시 보완):**
- 제외 사유에 해당하는 항목
- 필수 양식 누락 (1-20번 규칙)

**중간 (조속 보완):**
- 시험성적서 미비 (30번 규칙)
- 등기부등본 권리 관계

**낮음 (추후 보완 가능):**
- 세부 정보 확인 사항

### 3. 자주 묻는 질문

**Q: 시험성적서에 열전도율 시험만 있는데 통과하나요?**  
A: 아니요. 열전도율 시험은 검증 대상이 아닙니다. 반드시 열방출시험과 가스유해성 시험이 있어야 합니다.

**Q: 외벽이 석재인데 시험성적서가 필요한가요?**  
A: 석재의 경우 시험성적서는 생략 가능하고 납품확인서만 필요합니다. 시스템이 자동으로 인식합니다.

**Q: 여러 자재의 시험성적서를 제출해야 하나요?**  
A: 준공도면에서 추출된 모든 자재(외벽 마감재, 외벽 단열재, 필로티 마감재, 필로티 단열재)에 대해 시험성적서가 필요합니다.

**Q: 시험성적서가 여러 페이지인데 모두 업로드해야 하나요?**  
A: 네, 모든 페이지를 하나의 PDF로 업로드하세요. AI가 전체를 분석합니다.

## 💡 최적의 결과를 위한 팁

### PDF 품질
- **해상도**: 300 DPI 이상 권장
- **컬러**: 컬러 스캔 권장 (흑백도 가능)
- **페이지**: 여백 최소화

### 시험성적서 특히 주의
- 시험 항목 제목이 명확히 보이도록 스캔
- 표가 잘리지 않도록 전체 페이지 스캔
- "열방출", "가스유해성" 키워드가 선명하게

### 파일 정리
- 같은 건물의 서류는 파일명에 공통 접두사 사용
  - 예: `강남구_테헤란로_123_주택매도신청서.pdf`
  - 예: `강남구_테헤란로_123_시험성적서.pdf`
- 이렇게 하면 시스템이 자동으로 그룹화합니다
